# Temple Run - Python Kivy Game

An endless runner game for Android built with Python and Kivy.

## How to Run

### On Termux (Android)
```bash
pip install kivy
python main.py
```

### Build APK (requires Buildozer)
```bash
buildozer android debug
```

## Controls
- **Swipe Left/Right** - Move between lanes
- **Swipe Up** - Jump over barriers
- **Swipe Down** - Slide under beams

## Gameplay
- Run endlessly through a procedurally generated corridor
- Dodge obstacles: walls (change lane), barriers (jump), beams (slide)
- Collect coins for bonus points
- Speed increases over time
